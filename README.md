# BankDatabase

**BankDatabase** is a learning/test project of a bank database, built with C# using SQLite and .NET.  
The project allows you to work with banking data: add, delete, update records, and monitor the state of the database.

> ⚠️ The project can be improved in several ways, but overall it demonstrates the basic functionality of a bank database.

---

## 📂 Main Features

- Management of client and account data
- Adding, deleting, and updating records
- Monitoring the current state of the database
- Basic support for a web interface

---

## 🛠️ Technologies

- C# / .NET
- SQLite (file-based database)
- Entity Framework (for database operations)
- Visual Studio (for running and development)
